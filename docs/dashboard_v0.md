# ScrollGuard Dashboard v0

Run:

./scripts/run_dashboard.sh

Open in browser:

http://localhost:8501

Dashboard sections:

- Gate A CT crop
- Month 3 feature contact sheet
- Month 4 quality/risk contact sheet
- Review-priority overlay
- Review-priority ranking table
- Top-5 evidence package

This dashboard uses local ScrollGuard outputs only. It does not contact the Vesuvius server.
